# 🚀 Stop Searching. Start Learning.

> Solving **Resource FOMO** by helping students spend less time searching and more time learning.

## 📖 About

Students often spend hours comparing YouTube playlists, blogs, Reddit discussions, and AI recommendations before starting to learn.

Our platform solves this **Resource FOMO** by recommending the right learning resource based on the learner's profile.
